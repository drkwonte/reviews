## 1. 개요 (Overview)

### 1.1 프로젝트 요약

"nextime"은 틀린 문제를 촬영/업로드하면 AI가 수식과 텍스트를 정확히 읽어내고, 체계적인 복습 관리를 도와주는 **지능형 오답노트 에이전트**이다. **v2.4에서는 사용자의 학습 확장에 따라 인터페이스가 스스로 진화하는 '데이터 자생적 시스템'을 구축하여 확장성을 극대화했다.**

### 1.2 핵심 가치

- **데이터 자생적 UI (New, v2.4)**: 새로운 과목이 등록되면 버튼과 차트가 알아서 생성되는 지능형 대응.
- **실전 증명형 마케팅 (v2.3)**: 가짜 기능이 아닌, 실제 서비스의 강력한 통계와 데이터 분석 역량을 랜딩 페이지에서 직접 증명.
- **스마트 정렬 (Smart Sort)**: 파일명 숫자를 인식하는 지능형 이미지 정렬로 OCR의 맥락 보존.
- **최적화된 UX**: 다크 모드, 스켈레톤 UI, 글래스모피즘 헤더를 통한 프리미엄 시각 경험.

## 3. 기능 상세 (Features)

### 3.1 마케팅 및 접근 제어 (v2.3 추가)

- **하이엔드 랜딩 페이지**: 서비스의 핵심 가치(기록, AI 등록, 입체적 복습)를 실제 앱 화면과 리얼 데이터를 통해 시각화.
- **상태 기반 라우팅**: 로그인 유저가 앱 접근 시 랜딩 페이지를 거치지 않고 대시보드로 즉각 이동하여 생산성 유지.

### 3.9 지능형 데이터 동기화 (v2.4 완료)

- **Dynamic Filters**: DB에 존재하는 모든 유니크한 과목명을 실시간 추출하여 오답노트 목록 필터 버튼 자동 생성.
- **Zero-Count Insight**: 복습 기록이 없는 신규 과목도 차트에서 누락되지 않도록 전체 과목 리스트 기반의 통계 엔진 구축.
