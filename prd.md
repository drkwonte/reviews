## 1. 개요 (Overview)

### 1.1 프로젝트 요약

"nextime"은 시험에서 틀린 문제를 사진/스캔으로 업로드하면 원본 이미지 저장 + OCR 텍스트 변환을 동시에 수행하여 체계적으로 오답을 관리하고, AI가 해설을 생성하며, 지능형 통계를 통해 학습 성장을 관리하는 차세대 오답노트 플랫폼이다.

### 1.2 핵심 가치

- **스토리텔링 랜딩 (New, v2.3)**: 기술을 모르는 사용자도 서비스의 위력을 느낄 수 있는 고밀도 비주얼 쇼케이스 제공
- **지능형 분석 대시보드 (New, v2.3)**: 누적 추세, 과목별 분포, 복습 정답률 등을 시각화하여 학습의 '증명'을 제시
- **데이터 자생적 UI (New, v2.3)**: 사용자가 추가한 과목(예: 물리)을 시스템이 스스로 감지하여 차트와 필터 버튼을 자동 생성
- **스마트 정렬 및 병합**: 여러 장의 이미지를 순서대로 자동 정렬/병합하여 OCR 분석의 맥락 무결성 확보
- **이미지 중심의 기록**: 문제와 풀이의 핵심은 이미지이다. OCR 텍스트는 강력한 보조 데이터로 활용된다.

## 3. 기능 상세 (Features)

### 3.1 프리미엄 랜딩 페이지 (v2.3 신설)

- **AI 쇼케이스**: 실제 앱의 문제 등록 -> OCR -> 수식 시각화 -> 해설 생성 과정을 4단계 애니메이션으로 시각화한 리얼 월드 섹션
- **마스터 시스템**: 학습 캘린더 및 복습 지표를 직관적으로 보여주어 서비스 가입 후의 변화를 예고
- **Auth Redirect**: 로그인 상태 감지 시 랜딩 대신 대시보드(`/dashboard`)로 즉시 안내하는 사용자 경험 최적화

### 3.9 지능형 분석 및 자동화 (v2.3 업데이트)

- **과목 감지 엔진**: '국어/수학/영어'로 제한된 하드코딩 필터를 폐기하고, DB에서 유니크한 과목명을 실시간으로 추출하여 UI(차트, 필터 버튼)를 동적 생성
- **6종 마이크로 분석 차트**:
  1. 과목별 오답 분포 (Bar)
  2. 오늘의 활동성 현황 (Gauge)
  3. 누적 성장 추이 (Line)
  4. 학습 실천 캘린더 (Heatmap)
  5. 과목별 복습 회수 (Bar)
  6. 복습 정답률 분석 (Bar)
